# Latex
Based on https://github.com/latex-g7-32/latex-g7-32 

## To build Latex Paper on Windows
Start Docker image `aergus/latex` pass the name of instance to the .bat script
```
./build_paper.bat NAME_OF_DOCKER_INSTANCE
```
