\chapter*{\centerline{\bf Висновки}}\label{vysnovky}

Рандомізовані методи знаходять широке поле для застосувань у різних галузях діяльності людини, тож дослідження рандомізованих методів комп'ютерної алгебри є актуальною задачею.

В даній  роботі було розглянуто рандомізовані методи для знаходження елементів групових алгебр, які мають задані властивості, для деяких задач теорії груп та теорії чисел. До результатів дипломної роботи бакалавра можна віднести:

\begin{enumerate}[noitemsep,partopsep=0pt,topsep=0pt,parsep=0pt]
\item Було зроблено огляд рандомізованих методів комп'ютерної алгебри;
\item Створено модуль для системи комп'ютерної алгебри \textsf{GAP} для роботи із рандомізованими алгоритмами;
\item Розроблені алгоритми протестовано на чисельних прикадах.
\end{enumerate}
